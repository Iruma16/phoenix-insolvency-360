# Archivos Raw - Ley Concursal

Coloca aquí el archivo de texto con la Ley Concursal consolidada.

## Nombre del archivo

**Obligatorio**: `ley_concursal_consolidada.txt`

## Formato esperado

El archivo debe contener el texto completo de la Ley Concursal, con artículos numerados claramente:

```
Art. 1
Texto del artículo 1...

Art. 2
Texto del artículo 2...

...
```

## Origen

Obtén el texto consolidado desde:
- BOE: https://www.boe.es/buscar/act.php?id=BOE-A-2003-14086
- Texto consolidado oficial

## ⚠️ IMPORTANTE

- NO subas archivos automáticamente
- Verifica que el texto es el correcto antes de ingerir
- Actualiza `metadata.json` después de cualquier cambio
